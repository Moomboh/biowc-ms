# \<biowc-spectrum>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation

```bash
npm i biowc-spectrum
```

## Usage

```html
<script type="module">
  import 'biowc-spectrum/biowc-spectrum.js';
</script>

<biowc-spectrum></biowc-spectrum>
```
